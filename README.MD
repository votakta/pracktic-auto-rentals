#  Информационная система управления арендой легковых автомобилей

## О проекте
Разработан в рамках учебной практики. Представляет собой веб-приложение для аренды автомобилей с клиент-серверной архитектурой.

## Технологии

### Backend
- **Node.js** — среда выполнения
- **Express** — веб-фреймворк
- **SQLite** — база данных
- **JWT** — авторизация
- **bcrypt** — хеширование паролей

### Frontend
- **React** — библиотека для UI
- **React Router** — маршрутизация
- **Axios** — HTTP-запросы
